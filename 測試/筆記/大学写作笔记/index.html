第一章 写作活动概论

写作的意义

  1. 写作活动是个体智力和精神发育的途径
  2. 写作活动是个体精神生活的方式
  3. 写作能力是个人的职业发展的需要

写作活动的两种类型

  1. 实际应用性的文章写作

    一般有固定的写作模式、套路和写作规范。

    大都强调社会公共价值准则，不要求甚至是尽量避免写作者的个性化表达。不允许使用抒情、描写之类的表达方式和语态。

  2. 个人表达性质的文章写作

    能以写作者个体的独家经验，个性见解而为读者提供其所不具备的东西，对读者具有极明显的经验、知识方面的弥补价值或思想观念方面的参考、启发作用。

写作的源泉

  1. 生活感知能力

    a. 直接生活经验

    b. 间接生活经验

  2. 思想能力

思想的态度与方法

  1. 问题意识和怀疑精神
  2. 自由之思想，独立之精神
  3. 换位思考与多元化思维
  4. 创新思维
  5. 辩证思维

第二章 主题的设置与传达

写作的意图与形式

  1. 表达某种思想观点（议论文）
  2. 表现某种情感、情绪和某种意念或志趣，甚至是一种氛围（抒情文）
  3. 传达某种知识或信息（说明文）
  4. 讲述有趣的、有意义的故事或重要的事件（叙述文）
  5. 安排与处理某项事务（公文、文书）

主题有无价值或价值高低，一直是衡量文章价值和水平的主要标准。

立意的原则与要求

真诚、正确、深刻、新颖、集中

主题的传达方式

  1. 直接传达法：实用文（公文、新闻文、论文）、普通文（记叙文、议论文、说明文）

  2. 间接传达法：文艺性文章（小说、戏剧等）

    间接表意的方式：比兴手法、意象、比喻、典故、双关语等

第三章 材料的筛选、加工和组织

材料的类别

事实性材料、虚构性材料、观念性材料

材料的作用

  1. 材料触发写作意愿
  2. 材料产生写作主旨
  3. 材料表现主题思想
  4. 材料充实文章内容

获取和积累材料的途径

  1. 观察感受
  2. 调查采访
  3. 日常交流
  4. 阅读思考
  5. 上网检索

选材的基本原则

切题、真实、典型、新鲜

材料的加工

综合、概括、剪裁、嫁接、添枝加叶

材料的组织和编排

  1. 材料的表述
  2. 次序和穿插
  3. 详略处理
  4. 释义和说明

拟题纲的目的

解决文章的层次安排问题，使写作者对全文总体布局成竹在胸

结构安排的原则

  1. 要正确反映客观事物的发展规律和内外联系
  2. 要服从主题表达的需要
  3. 要适应不同文体的特点
  4. 要适应读者的阅读习惯和审美趣味

结构的要点

  1. 标题：贴切、简洁、吸引人
  2. 开头与结尾：自然结尾、束前结尾、推后结尾
  3. 层次和段落：并列式、递进式、总分式
  4. 过渡与照应：前后照应、内容与标题照应、行文中互相照应

第五章 语言文字的运用

文言、欧化语、方言俗语

  1. 文言：精炼含蓄、声韵优美、形式整齐
  2. 欧化语：注重逻辑性和分析性，讲求形式化，讲求语法
  3. 方言俗语：幽默鲜活、富于生活气息、具有极强表现力

基本要求：准确、简洁、明晰（改错题）※

看课本

常见的语言艺术风格

  1. 简练‹——›繁复

  2. 朴实‹——›绚丽

  3. 正统‹——›新奇

第六章 叙述的方法与艺术

叙述中的时间

故事时间、叙述时间

时间次序的处理

顺叙、倒叙、插叙（追叙、补叙、诠叙、预叙）、平叙

叙述角度

全知叙事（全知视角、外视角）：不受具体时间、空间的限制，还可以充分表达叙述者的主观倾向和评价。视野开阔，自由自在，适合表现广阔的社会生活画面；缺点是交代得太多太充分，压缩读者参与空间

限制叙事（限知视角、内视角）：由于叙述者所知有限，就可以产生悬念、意外等效果。

客观视角：只能讲述叙述者所见所听，不能进入人物心灵，也不能写叙述者自己的心理活动和主观意见，有助于制造客观化、神秘化、陌生化的效果。

人称使用

  1. 第一人称：

    线索清楚、简单方便

    颇具真实感

    主观性强便于抒情

    局限性：叙述内容不能超过 “我” 的耳闻目睹和亲身经历的范围；不能直接描写他人的心理活动。

    自我叙事与配角叙事

  2. 第三人称

常用叙述模式

第一人称限制叙事、第三人称限制叙事、第三人称客观叙事、第三人称全知叙事

情节设计的艺术

  1. 设置悬念
  2. 误会与巧合
  3. 突转与意外
  4. 重复与延宕

第七章 描写与抒情

一般出现在文艺性文体和一般性文体的写作之中，在应用文体中极少出现

描写的类型：按照对象

  1. 人物描写

    a. 肖像描写

    b. 动作描写

    c. 言语描写

    d. 心理描写

  2. 环境描写

    a. 自然风景描写

    b. 人文环境描写

  3. 场面描写

描写的类型：按照角度

  1. 正面描写

    白描：笔墨简省，重写意、传神

    细描：浓墨重彩，重写实、摹形

  2. 侧面描写

    烘托、反衬

抒情的方式与技巧

  1. 直接抒情

  2. 间接抒情

    a. 借叙事抒情

    b. 借描写抒情

    c. 借议论抒情

    d. 借说明抒情

第八章 说明与议论

说明的方法

  1. 定义说明
  2. 诠释说明
  3. 分类说明
  4. 分解说明
  5. 比较说明
  6. 举例说明
  7. 数据说明
  8. 图表说明

议论的三要素

论点、论据、论证

常见的论证方法

  1. 例证法
  2. 归纳法
  3. 演绎法
  4. 比较法
  5. 反证法

调查问卷

  1. 目的性原则
  2. 逻辑性原则
  3. 体谅被调查者的原则
  4. 便于处理性原则
  5. 合理的问卷长度原则
